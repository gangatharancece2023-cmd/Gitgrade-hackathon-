Repository Mirror – AI GitHub Evaluator

## Problem Statement
Recruiters struggle to judge a student’s GitHub repository quality objectively.
This project analyzes a GitHub repo and generates a score, summary, and personalized roadmap.

## Approach
1. Fetch public repository data using GitHub API
2. Evaluate code quality using explainable rule-based scoring
3. Generate mentor-like feedback and improvement roadmap

## Key Features
- Multi-dimensional evaluation
- Honest and actionable feedback
- Real-time analysis
- Beginner-friendly UI

## Tech Stack
- Python
- Streamlit
- GitHub REST API

## How to Run
pip install -r requirements.txt  
streamlit run app.py

## Real-World Use Case
Used by recruiters, mentors, and students to assess project quality instantly.
